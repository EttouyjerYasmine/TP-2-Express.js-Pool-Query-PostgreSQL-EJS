# TP Express.js – Gestion des Utilisateurs avec PostgreSQL et EJS

## Description

Ce TP consiste à créer une application web de **gestion des utilisateurs** en utilisant :

- **Node.js** avec **Express.js** pour le serveur
- **PostgreSQL** pour la base de données
- **EJS** pour le rendu des vues côté serveur
- **express-validator** pour la validation des données utilisateurs

L’application permet de **créer, lire, mettre à jour et supprimer (CRUD)** des utilisateurs.
